# kde_panel_colorizer_theme_importer
Move theme folders to /home/{user}/.config/panel-colorizer/presets

# How to use
Download main.py and create a dir for it(call it whatever you want)
make a folder nammed "Themes" in the same dir as main.py

execute main.py with python and input the index number of the theme you want to add as a preset(must be a folder)
